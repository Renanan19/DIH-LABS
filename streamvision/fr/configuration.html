Guide de Configuration IPTV

Apprenez à tirer le meilleur parti de StreamVision en quelques secondes.

Configuration via URL M3U

Le format M3U est le plus répandu pour les listes de lecture IPTV. Voici comment l'utiliser :

  • 1. Copiez votre URL : Récupérez le lien fourni par votre fournisseur de service.
  • 2. Ajoutez dans l'App : Cliquez sur le bouton "+" en haut à droite.
  • 3. Nommez votre liste : Donnez un nom (ex: "Ma TV") pour la retrouver facilement.
  • 4. Profitez : StreamVision télécharge et organise vos chaînes instantanément.

Configuration via Xtream Codes

L'API Xtream Codes offre une meilleure gestion des catégories Films et Séries.

  • Serveur : L'adresse URL (ex: http://nom-du-serveur.com:8080).
  • Identifiants : Saisissez votre Username et Password.
  • Auto-Sync : StreamVision synchronise automatiquement les affiches et résumés.
